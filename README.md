# Python Sales Data Analysis Project

## Objective
Analyze retail sales data using Python to identify sales trends, profitability, and business insights.

## Tools Used
- Python
- Pandas
- Matplotlib

## Analysis Performed
- Sales and Profit Analysis
- Category-wise Sales and Profit
- Region-wise Performance
- Year-wise Sales Trend
- Discount Impact on Profit
- Identification of Loss-Making Products

## Key Insights
- Technology category generates the highest sales, but profit varies across categories  
- Some products result in consistent losses  
- Higher discounts tend to reduce profit  
- Sales increased significantly over time, especially after 2015
