# Python Sales Data Analysis Project

## Objective
Analyze retail sales data using Python to identify category, region, and yearly sales trends.

## Tools Used
- Python
- Pandas
- Matplotlib

## Key Insights
- Technology category generated the highest sales
- West region contributed the most revenue
- Sales increased significantly by 2017
- Python was used to clean, group, and visualize the data
